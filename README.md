## üö¶ Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/AryanSONI00/CoEdit.git
cd CoEdit
```

### 2. Start the Server

```bash
cd server
npm install
npm run dev
```

### 3. Start the Client

```bash
cd ../client
npm install
npm run dev
```

### 4. Open in Browser

Visit [http://localhost:5173](http://localhost:5173) (or the port shown in your terminal) to start collaborating!
